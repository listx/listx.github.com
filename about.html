Recommended Software

I use free/open-source software almost exclusively as much as possible. Here are some of them I heartily recommend.

  • Arch Linux - my favorite OS
  • dm-crypt/LUKS - my choice for block device encryption
  • Terminus - my favorite monospaced font
  • rxvt-unicode (aka urxvt) - the only sane Unicode-aware terminal emulator
  • Xmonad - the crash-proof tiled window manager
  • Vim - an advanced text editor
  • Emacs - an even more advanced text editor
  • evil - an Emacs addon that makes it behave like Vim
  • GnuPG (aka gpg) - a way to ensure secure communications
  • Firefox - the most mature and stable customizable web browser
  • Pentadactyl - addon to make Firefox behave like Vim
  • Texlive - the modern TeX environment (XeTeX being my preferred engine)
  • Lilypond - if TeX is the sophisticated typesetting system, then Lilypond is the sophisticated sheet-music-setting system
  • Blender - an advanced 3D graphics program
  • Luxrender - an advanced photorealistic renderer, which pairs nicely with Blender via luxblend
